# Playing with MNIST using Numpy

1. Create a new repository with this project and solve it.

2. You will need to install TensorFlow to download the MNIST dataset.

> [!TIP]
> Research and try the [main package managers fo creating a virtual environment](https://github.com/avidaldo/ia25/blob/master/setup/setup.md#virtual-environments) and installing Keras.


3. Solve the exercises in [numpy-mnist.ipynb](./numpy-mnist.ipynb) using NumPy only.

> [!TIP]
> Study the [notes on Numpy](https://github.com/avidaldo/ia25/blob/master/README.md#basic-setup-and-tools) to find the best solutions.

4. Any time you want to save your progress, commit and push your changes to your forked repository.

5. Any time you want to submit your solution for review, create a pull request to this repository.

> [!TIP]
> The project includes certain tests that you can run to check if your code works correctly. You can run them with `pytest` from the terminal.
> Additionally, they run automatically in GitHub Actions when you push, so you can see if your code passes the tests in the "Actions" tab of your repository.
